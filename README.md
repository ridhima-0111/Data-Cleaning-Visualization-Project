# Data Cleaning & Visualization Project

## Objective

The objective of this project is to clean a raw sales dataset and generate meaningful insights through data visualization.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Dataset

Superstore Sales Dataset

## Data Cleaning Performed

* Checked missing values
* Removed duplicate records
* Filled missing values
* Processed and cleaned the dataset

## Visualization Created

* Sales by Category

## Output Files

* cleaned_data.csv
* sales_chart.png

## Learning Outcome

This project helped in understanding data preprocessing, data cleaning, and data visualization using Python.
